    CryptoRunners is a collection of 1000 unique digital Runners. Each movement component and all combinations are created randomly using a generator developed Robert Norenberg. Each raw element is exported to a LQ PNG file; subsequently, each Cryptorunner is manually composed in a HQ 480x480 GIF format. The uniqueness is controlled through an SQL database.
